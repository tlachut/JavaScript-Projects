
# Symulacja losowania totolotka

Ten projekt umożliwia użytkownikowi wirtualne uczestnictwo w losowaniu loterii lotto. Głównym celem jest wytypowanie 6 liczb spośród zakresu od 1 do 49.

# Instrukcja

1. Wprowadź swoje typy liczb od 1 do 49 w pola oznaczone numerami 1-6.
2. Możesz również skorzystać z przycisku "Losowe", aby losowo wygenerować liczby.
3. Upewnij się, że wprowadzone liczby są unikatowe, nie mogą się powtarzać.
4. Po wprowadzeniu liczb, naciśnij przycisk "Zatwierdź", aby potwierdzić swoje wybory.
5. Program przeprowadzi losowanie 6 liczb i wyświetli wyniki.
6. Przycisk "Reset" pozwala zresetować wybory użytkownika i wylosowane liczby.
